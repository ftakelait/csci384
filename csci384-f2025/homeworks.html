Submission Guidelines

All assignments and projects should be submitted through Blackboard unless otherwise specified. Each submission should include:

  • Complete source code with comments
  • README file explaining how to run your code
  • Results and analysis
  • Performance evaluation and comparisons
  • Any additional files required

Collaboration Policy

You are encouraged to discuss assignment and project problems with your classmates. However, you must write your own code and solutions. Copying code from others or from the internet is not allowed and will result in academic dishonesty.

Late Policy

Assignments and projects submitted after the deadline will receive a 10% penalty per day, up to a maximum of 3 days. After 3 days, no credit will be given.

Grading

Each assignment and project will be graded on:

  • Correctness of implementation (60%)
  • Code quality and documentation (20%)
  • Analysis and discussion of results (20%)

Getting Help

If you need help with assignments or projects:

  • Check Blackboard for announcements and discussions
  • Attend office hours
  • Email the instructor or TAs for specific questions
  • Use the course discussion forum on Blackboard
