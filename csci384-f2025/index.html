Course description

A study of algorithms and application of AI. The topics include agent theory, problem-solving with the search, constraint satisfaction problem, game, knowledge-based system, reasoning and machine learning which are widely applicable to design of an intelligent system, data science and mining, information retrieval, pathfinding and classification, etc.

Pre-requisite: CSCI 242 or equivalent (with permission of instructor).

Class format

Each class will be divided into

  • Lecture
  • Practicum/Lab

There will be a 10-minute break after the lecture.

During the Practicum/Lab segment of the class, we will focus on technical (coding) skills and provide project guidance. You will be expected to work on your projects during this time, so please bring your laptop.

Programming Assignments

There will be 4 programming assignments throughout the semester:

  1. A* Search Algorithm: Implementation of A* pathfinding algorithm
  2. Hidden Markov Model (HMM): Weather prediction using HMM
  3. Naive Bayes Classifier: Spotify music classification
  4. Bayesian Network: Bioinformatics data analysis

All assignment code and repositories are available on GitHub.

Major Projects

There will be 2 major projects throughout the semester:

  1. CSP Scheduling Problem (Weeks 5-8): Constraint Satisfaction Problem implementation for scheduling
  2. Markov Decision Process (MDP) (Weeks 8-15): Policy iteration and value iteration for robot navigation

All project code and repositories are available on GitHub.

Please see the projects page for more details.
